# 列表样式 list-style-type

列表样式是针对，有序列表和无序列表。

```list-style-type:取值```

## 有序列表的情况下

取值有以下几种:

* decimal 数字(默认值)
* lower-roman 小写罗马数字
* upper-roman 大写罗马数字
* lower-alpha 小写英文字母
* upper-alpha 大写英文字母

## 无序列表的情况下

取值有以下几种:

* disc 实心圆(默认值)
* circle 空心圆
* square 正方形

## 去除修饰

当list-style-type:none时无论有序列表还是无序列表都可以去除前面的修饰。
